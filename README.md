# Fusion Five - University Management System

A comprehensive web-based university Management System built with HTML, CSS, and JavaScript. This project provides separate portals for administrators, teachers, students, and library management.

1st semester final project based on HTML CSS Javascripts 


## Project Overview

**Fusion Five** is a complete university management solution that allows educational institutions to manage:
- Student records and information
- Teacher staff management
- Library book collection
- Administrative operations

## Features

### 1. Admin Portal (`admin.html`)
- Secure login system
- Add new administrators
- View all administrators
- Search admin by ID
- Update admin information
- Delete admin records

### 2. Student Portal (`student.html`)
- Student login authentication
- Add new student records
- View all students
- Search student by roll number
- Update student data
- Delete student records
- Track student information (name, father name, contact, age, class, address)

### 3. Teacher Portal (`teacher.html`)
- Teacher login authentication
- Add new teacher records
- View all teachers
- Search teacher by ID
- Update teacher information
- Delete teacher records
- Track teacher details (subject, qualification, contact, email)

### 4. Library Management (`library.html`)
- Add new books
- View all books
- Search books by ID
- Update book data
- Delete books
- Track book information (title, author, genre)
- Statistics dashboard (total books, genres, authors)

### 5. Main Dashboard (`index.html`)
- Central navigation hub
- Quick access to all portals
- Exit option

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Styling**: Custom Inline CSS with animations and gradients
- **No Backend**: Data stored in JavaScript arrays (simulated database)
